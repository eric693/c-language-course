1. C語言簡介

為什麼學習C語言？

C語言是一門通用的程式語言，由Dennis Ritchie於1972年在貝爾實驗室開發。它具有以下特點：

  • 效能優異：編譯後的程式執行效率高，接近組合語言
  • 可移植性強：只要有C編譯器，程式可以在不同平台上執行
  • 底層控制：可以直接操作記憶體與硬體
  • 基礎語言：許多現代語言（C++、Java、C#）都源自於C
  • 系統程式：作業系統、編譯器、驅動程式多用C開發

C語言的應用領域

  • 作業系統開發（Linux、Unix）
  • 嵌入式系統與物聯網
  • 系統程式與工具程式
  • 遊戲引擎與圖形處理
  • 資料庫系統
  • 網路協定與伺服器

2. 開發環境建置

Windows 環境

方案一：Visual Studio Community（推薦初學者）

  1. 前往 Visual Studio 官網下載 Community 版本（免費）
  2. 安裝時選擇「使用C++的桌面開發」工作負載
  3. 建立新專案時選擇「空專案」或「主控台應用程式」
  4. 新增 .c 檔案即可開始撰寫C程式

方案二：Code::Blocks + MinGW

  1. 下載 Code::Blocks with MinGW 安裝包
  2. 執行安裝程式，選擇完整安裝
  3. 開啟 Code::Blocks，建立新專案（Console Application）
  4. 選擇 C 語言專案即可開始開發

方案三：VS Code + MinGW（進階使用者）

  1. 安裝 Visual Studio Code
  2. 安裝 MinGW-w64 編譯器
  3. 在 VS Code 中安裝 C/C++ 擴充套件
  4. 設定編譯與偵錯環境

Linux 環境

# Ubuntu / Debian 系統 sudo apt update sudo apt install build-essential # Fedora / CentOS 系統 sudo dnf install gcc gcc-c++ make # 驗證安裝 gcc --version make --version

macOS 環境

# 安裝 Xcode Command Line Tools xcode-select --install # 或安裝完整的 Xcode（從 App Store） # 驗證安裝 gcc --version clang --version

基本編譯流程

# 1. 撰寫程式碼（例如：hello.c） # 2. 編譯程式 gcc hello.c -o hello # 3. 執行程式 ./hello # Linux / macOS hello.exe # Windows # 編譯選項說明： # -o : 指定輸出檔案名稱 # -Wall : 啟用所有警告訊息 # -g : 產生除錯資訊 # -O2 : 最佳化等級 2 # 完整編譯範例 gcc -Wall -Wextra -g hello.c -o hello
學習建議： 建議初學者從簡單的文字編輯器（如 Notepad++、Sublime Text）加上命令列編譯開始， 這樣可以更深入了解編譯流程。待熟悉後再使用完整的 IDE。

3. 第一個C程式

Hello World 程式

#include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }

程式結構解析

程式碼 說明
#include <stdio.h> 引入標準輸入輸出標頭檔，提供 printf、scanf 等函數
int main() 主函數，程式執行的入口點。int 表示函數返回整數
{ ... } 大括號定義函數或程式區塊的範圍
printf(...) 輸出函數，將文字顯示到標準輸出（通常是螢幕）
\n 跳脫字元，代表換行
return 0; 返回 0 給作業系統，表示程式正常結束
; 分號表示敘述結束（每個敘述都要加分號）

編譯與執行

# 將程式存成 hello.c # 編譯 gcc hello.c -o hello # 執行 ./hello # 輸出結果： Hello, World!
重要觀念：
  • C語言區分大小寫：main 和 Main 是不同的
  • 每個敘述必須以分號結束
  • 縮排不是必需的，但能增加程式可讀性
  • 標頭檔包含要放在程式最前面

4. 基本資料型態與變數

基本資料型態

型態 大小 範圍 用途
char 1 byte -128 ~ 127 字元或小整數
unsigned char 1 byte 0 ~ 255 無號字元
short 2 bytes -32,768 ~ 32,767 短整數
int 4 bytes -2,147,483,648 ~ 2,147,483,647 整數
long 4/8 bytes 依平台而定 長整數
float 4 bytes 約 7 位有效數字 單精度浮點數
double 8 bytes 約 15 位有效數字 雙精度浮點數

變數宣告與初始化

#include <stdio.h> int main() { // 變數宣告 int age; float height; char grade; // 變數賦值 age = 20; height = 175.5; grade = 'A'; // 宣告時初始化（推薦） int score = 95; double pi = 3.14159; char initial = 'J'; // 同時宣告多個變數 int x, y, z; int a = 1, b = 2, c = 3; // 輸出變數值 printf("Age: %d\n", age); printf("Height: %.1f cm\n", height); printf("Grade: %c\n", grade); return 0; }

變數命名規則

  • 只能包含英文字母、數字和底線 (_)
  • 不能以數字開頭
  • 不能使用C語言的保留字（如 int, float, if, while 等）
  • 區分大小寫（age 和 Age 是不同的變數）
  • 建議使用有意義的變數名稱（如 student_count 而非 sc）
命名慣例：
  • 駝峰式命名：studentCount, firstName
  • 底線命名：student_count, first_name（C語言較常用）
  • 常數：全大寫，如 MAX_SIZE, PI

變數作用域

#include <stdio.h> // 全域變數（在所有函數外宣告） int global_var = 100; int main() { // 區域變數（在函數內宣告） int local_var = 50; printf("Global: %d\n", global_var); // 可以存取 printf("Local: %d\n", local_var); // 可以存取 { // 區塊內的變數 int block_var = 25; printf("Block: %d\n", block_var); } // printf("%d", block_var); // 錯誤！block_var 超出作用域 return 0; }

5. 運算符號

算術運算符

#include <stdio.h> int main() { int a = 10, b = 3; printf("a + b = %d\n", a + b); // 加法：13 printf("a - b = %d\n", a - b); // 減法：7 printf("a * b = %d\n", a * b); // 乘法：30 printf("a / b = %d\n", a / b); // 除法：3（整數除法） printf("a %% b = %d\n", a % b); // 餘數：1 // 浮點數除法 float c = 10.0, d = 3.0; printf("c / d = %.2f\n", c / d); // 3.33 return 0; }

賦值運算符

int x = 10; x += 5; // x = x + 5; 結果：15 x -= 3; // x = x - 3; 結果：12 x *= 2; // x = x * 2; 結果：24 x /= 4; // x = x / 4; 結果：6 x %= 5; // x = x % 5; 結果：1

遞增與遞減運算符

#include <stdio.h> int main() { int a = 5; // 後遞增：先使用再加 1 printf("%d\n", a++); // 輸出 5，然後 a 變成 6 printf("%d\n", a); // 輸出 6 // 前遞增：先加 1 再使用 printf("%d\n", ++a); // a 先變成 7，然後輸出 7 // 遞減運算符類似 a--; // a 變成 6 --a; // a 變成 5 return 0; }

關係運算符

運算符 說明 範例 結果
== 等於 5 == 5 1 (真)
!= 不等於 5 != 3 1 (真)
> 大於 5 > 3 1 (真)
< 小於 5 < 3 0 (假)
>= 大於等於 5 >= 5 1 (真)
<= 小於等於 5 <= 3 0 (假)

邏輯運算符

#include <stdio.h> int main() { int a = 1, b = 0; // && (AND)：兩邊都為真時才是真 printf("%d\n", a && b); // 0 (假) printf("%d\n", a && a); // 1 (真) // || (OR)：至少一邊為真就是真 printf("%d\n", a || b); // 1 (真) printf("%d\n", b || b); // 0 (假) // ! (NOT)：反轉真假值 printf("%d\n", !a); // 0 (假) printf("%d\n", !b); // 1 (真) return 0; }
常見錯誤：
  • 使用 = 而非 == 做比較：if (x = 5) 是賦值，不是比較！
  • 整數除法會捨棄小數部分：10 / 3 結果是 3，不是 3.333
  • 除數為零會導致程式崩潰

6. 標準輸入輸出

printf() - 格式化輸出

#include <stdio.h> int main() { int age = 25; float height = 175.5; char grade = 'A'; char name[] = "張三"; // 基本輸出 printf("年齡：%d\n", age); printf("身高：%.1f cm\n", height); printf("等級：%c\n", grade); printf("姓名：%s\n", name); // 多個變數輸出 printf("%s 今年 %d 歲，身高 %.1f 公分\n", name, age, height); // 格式控制 printf("%5d\n", 123); // 右對齊，寬度5： 123 printf("%-5d\n", 123); // 左對齊，寬度5：123 printf("%05d\n", 123); // 補零：00123 printf("%8.2f\n", 3.14); // 寬度8，2位小數： 3.14 return 0; }

printf() 格式符號

符號 說明 範例
%d, %i 帶符號十進位整數 printf("%d", 123);
%u 無符號十進位整數 printf("%u", 123);
%f 浮點數 printf("%.2f", 3.14);
%e, %E 科學記號 printf("%e", 1234.5);
%c 字元 printf("%c", 'A');
%s 字串 printf("%s", "Hello");
%p 指標位址 printf("%p", &var);
%x, %X 十六進位 printf("%x", 255);
%o 八進位 printf("%o", 64);
%% 輸出 % 符號 printf("100%%");

scanf() - 格式化輸入

#include <stdio.h> int main() { int age; float height; char name[50]; // 讀取整數 printf("請輸入年齡："); scanf("%d", &age); // 讀取浮點數 printf("請輸入身高："); scanf("%f", &height); // 讀取字串（遇到空白會停止） printf("請輸入姓名："); scanf("%s", name); // 注意：字串不用加 & // 輸出結果 printf("\n您的資料：\n"); printf("姓名：%s\n", name); printf("年齡：%d 歲\n", age); printf("身高：%.1f 公分\n", height); return 0; }

scanf() 使用注意事項

重要提醒：
  • 一般變數前要加 & 符號（取址運算符），但陣列名稱本身就是位址，不用加 &
  • 格式符號要與變數型態對應，否則會產生錯誤
  • %s 讀取字串時遇到空白、Tab 或換行就會停止
  • 要讀取含空格的字串，使用 fgets() 函數較佳
  • 輸入前清除緩衝區可避免一些問題

其他輸入輸出函數

#include <stdio.h> int main() { char ch; // getchar() - 讀取單一字元 printf("請輸入一個字元："); ch = getchar(); // putchar() - 輸出單一字元 printf("你輸入的字元是："); putchar(ch); putchar('\n'); // puts() - 輸出字串（自動加換行） puts("這是一行文字"); // gets() - 讀取一行（已不建議使用，有安全問題） // 建議使用 fgets() 取代 return 0; }

7. 程式註解與編碼風格

註解方式

#include <stdio.h> // 單行註解：用於簡短說明 // 這種註解只影響到該行結束 /* * 多行註解：用於較長的說明 * 可以跨越多行 * 常用於函數或模組的說明 */ int main() { int x = 10; // 行尾註解 /* * 這是一段較複雜的運算 * 需要詳細說明 */ int result = x * 2 + 5; return 0; }

良好的編碼風格

1. 有意義的變數命名

// 不好的命名 int a, b, c; float x; // 好的命名 int student_count, total_score, average; float temperature_celsius;

2. 適當的縮排

// 不好的風格 int main() { int x=10; if(x>5){ printf("大於5"); } return 0; } // 好的風格（建議使用 4 個空格縮排） int main() { int x = 10; if (x > 5) { printf("大於5\n"); } return 0; }

3. 適當的空白與換行

// 運算符兩側加空格 int sum = a + b; // 好 int sum=a+b; // 不好 // 逗號後面加空格 printf("%d, %d", x, y); // 好 printf("%d,%d",x,y); // 不好 // 函數間加空行 void func1() { // ... } void func2() { // ... }

4. 常數使用 #define

#include <stdio.h> // 定義常數（全大寫命名） #define PI 3.14159 #define MAX_SIZE 100 #define BUFFER_LENGTH 256 int main() { double radius = 5.0; double area = PI * radius * radius; printf("圓面積：%.2f\n", area); return 0; }
編碼風格建議：
  • 保持一致的縮排方式（建議 4 個空格）
  • 每行程式碼不要太長（建議不超過 80-100 字元）
  • 適當使用空行分隔邏輯區塊
  • 函數和變數命名要有意義
  • 重要的邏輯要加註解說明
  • 避免過度註解明顯的程式碼

8. 完整範例程式

範例：簡易計算機

#include <stdio.h> int main() { float num1, num2, result; char operator; // 顯示標題 printf("===== 簡易計算機 =====\n\n"); // 輸入第一個數字 printf("請輸入第一個數字："); scanf("%f", &num1); // 輸入運算符號 printf("請輸入運算符號 (+, -, *, /)："); scanf(" %c", &operator); // 注意：%c 前面的空格 // 輸入第二個數字 printf("請輸入第二個數字："); scanf("%f", &num2); // 執行運算 if (operator == '+') { result = num1 + num2; printf("\n%.2f + %.2f = %.2f\n", num1, num2, result); } else if (operator == '-') { result = num1 - num2; printf("\n%.2f - %.2f = %.2f\n", num1, num2, result); } else if (operator == '*') { result = num1 * num2; printf("\n%.2f * %.2f = %.2f\n", num1, num2, result); } else if (operator == '/') { if (num2 != 0) { result = num1 / num2; printf("\n%.2f / %.2f = %.2f\n", num1, num2, result); } else { printf("\n錯誤：除數不能為零！\n"); } } else { printf("\n錯誤：無效的運算符號！\n"); } return 0; }

範例：溫度轉換程式

#include <stdio.h> int main() { float celsius, fahrenheit; printf("===== 溫度轉換程式 =====\n\n"); printf("請輸入攝氏溫度："); scanf("%f", &celsius); // 轉換公式：F = C × 9/5 + 32 fahrenheit = celsius * 9.0 / 5.0 + 32.0; printf("\n攝氏 %.2f 度 = 華氏 %.2f 度\n", celsius, fahrenheit); return 0; }

本週練習題

基礎練習

  1. 個人資料輸出
    撰寫程式輸出你的姓名、學號、年齡、身高、體重等資訊，使用適當的格式化輸出。
  2. 變數交換
    撰寫程式，讀取兩個整數 a 和 b，然後交換它們的值並輸出。
    提示：需要使用第三個變數作為暫存。
  3. 圓形計算
    讀取圓的半徑，計算並輸出圓周長和圓面積。
    公式：周長 = 2πr，面積 = πr²

進階練習

  1. 購物結帳
    撰寫程式讀取商品單價和數量，計算小計，然後讀取折扣率（如 0.9 表示九折）， 計算折扣後金額，最後讀取顧客付款金額，計算找零。
  2. 時間轉換
    讀取秒數，將其轉換為「小時:分鐘:秒」的格式輸出。
    例如：輸入 3665，輸出「1 小時 1 分鐘 5 秒」
  3. BMI 計算器
    讀取身高（公分）和體重（公斤），計算並輸出 BMI 值。
    公式：BMI = 體重(kg) / (身高(m))²

挑戰練習

  1. 單位換算程式
    撰寫一個程式，讓使用者選擇要進行的換算類型（長度、重量、溫度等）， 然後輸入數值進行換算。例如：公分轉公尺、公斤轉磅、攝氏轉華氏等。
練習提示：
  • 先在紙上規劃程式流程和需要的變數
  • 確保變數型態選擇正確（整數 vs 浮點數）
  • 注意整數除法與浮點數除法的差異
  • 適當使用括號確保運算順序正確
  • 測試時使用不同的輸入值驗證程式正確性
