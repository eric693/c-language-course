1. C語言簡介

C語言的歷史

C語言由 Dennis Ritchie 於 1972 年在貝爾實驗室開發，最初是為了開發 UNIX 作業系統。經過50多年的發展，C語言依然是最重要的程式語言之一。

為什麼學習C語言？

  • 效能優秀：直接操作記憶體，執行速度快，適合開發需要高效能的應用
  • 系統程式設計：作業系統、驅動程式、編譯器的主要開發語言
  • 嵌入式系統：廣泛應用於硬體控制、IoT裝置、單晶片開發
  • 基礎語言：許多現代語言（C++、Java、C#、JavaScript）都受到C語言影響
  • 跨平台：可在各種作業系統和硬體平台上編譯執行
  • 豐富的函式庫：有大量成熟的開源函式庫可使用
  • 深入理解電腦：學習C語言能更了解電腦底層運作原理

C語言的應用領域

  • 作業系統：Linux、Unix、Windows核心部分
  • 嵌入式系統：單晶片、感測器、物聯網裝置
  • 資料庫系統：MySQL、PostgreSQL、SQLite
  • 編譯器與直譯器：GCC、Python直譯器
  • 遊戲引擎：Unreal Engine、Unity底層
  • 網路協定：TCP/IP協定堆疊實作
  • 科學計算：數值分析、模擬計算
  • 圖形處理：OpenGL、圖像處理函式庫

C語言的特點

優點：

  • 執行效率高，接近組合語言的效能
  • 語法簡潔，核心語法不複雜
  • 可移植性強，標準化程度高
  • 能直接操作記憶體和硬體
  • 豐富的運算符和資料型態

需要注意：

  • 記憶體管理需要手動處理
  • 沒有內建的安全檢查機制
  • 指標概念需要時間理解
  • 除錯相對困難

2. 開發環境建置

Windows 環境

方法一：使用 MinGW-w64

  1. 下載 MinGW-w64
    • 訪問 MinGW-w64官網
    • 或從 SourceForge 下載
  2. 安裝步驟
    • 執行安裝程式
    • 選擇 Architecture: x86_64（64位元系統）
    • 選擇 Threads: posix
    • 選擇 Exception: seh
    • 記住安裝路徑（例如：C:\mingw64）
  3. 設定環境變數
    • 右鍵點擊「本機」→「內容」→「進階系統設定」
    • 點擊「環境變數」
    • 在「系統變數」中找到「Path」，點擊「編輯」
    • 新增：C:\mingw64\bin（根據你的安裝路徑）
    • 按「確定」儲存
  4. 驗證安裝
    • 開啟命令提示字元（Win + R，輸入 cmd）
    • 輸入：gcc --version
    • 輸入：g++ --version
    • 如果顯示版本資訊，表示安裝成功

方法二：使用 Dev-C++

  1. 下載 Dev-C++
    • 訪問 Dev-C++ 官網
    • 下載最新版本的安裝程式
  2. 安裝
    • 執行安裝程式，選擇繁體中文
    • 完整安裝（包含編譯器）
    • 安裝完成後啟動 Dev-C++
  3. 優點
    • 整合開發環境（IDE），介面友善
    • 內建編譯器，安裝簡單
    • 適合初學者使用
    • 支援專案管理

方法三：使用 Visual Studio Code

  1. 安裝 VS Code
    • 從官網下載安裝
  2. 安裝 C/C++ 擴充套件
    • 開啟 VS Code
    • 點擊左側擴充功能圖示
    • 搜尋「C/C++」，安裝 Microsoft 官方套件
    • 搜尋「Code Runner」安裝（方便執行程式）
  3. 搭配 MinGW
    • 需要先安裝 MinGW-w64（方法一）
    • VS Code 會自動偵測編譯器

基本編譯流程

# ========== 基本編譯 ========== # 1. 撰寫程式碼（例如：hello.c） # 使用任何文字編輯器撰寫 C 程式 # 2. 編譯程式 gcc hello.c -o hello # 3. 執行程式 ./hello # Linux / macOS hello.exe # Windows（或直接輸入 hello） # ========== 編譯選項說明 ========== # -o filename : 指定輸出檔案名稱 gcc hello.c -o myprogram # -Wall : 啟用所有常見警告訊息 gcc -Wall hello.c -o hello # -Wextra : 啟用額外的警告 gcc -Wall -Wextra hello.c -o hello # -g : 產生除錯資訊（用於 GDB） gcc -g hello.c -o hello # -O2 : 最佳化等級 2（提升執行效能） # -O0 : 不最佳化（預設，適合除錯） # -O1, -O2, -O3 : 不同等級的最佳化 gcc -O2 hello.c -o hello # -std=c11 : 指定C語言標準版本 # c89, c99, c11, c17 等 gcc -std=c11 hello.c -o hello # ========== 完整編譯範例 ========== # 開發階段：啟用所有警告、產生除錯資訊 gcc -Wall -Wextra -g hello.c -o hello # 發布階段：最佳化、移除除錯資訊 gcc -Wall -O2 hello.c -o hello # ========== 編譯多個檔案 ========== # 方法一：一次編譯所有檔案 gcc -Wall main.c utils.c -o myprogram # 方法二：分別編譯成目標檔，再連結 gcc -c main.c # 產生 main.o gcc -c utils.c # 產生 utils.o gcc main.o utils.o -o myprogram # 連結 # ========== 常見編譯問題 ========== # 如果出現「command not found」錯誤： # 檢查是否已安裝 GCC which gcc gcc --version # 如果編譯成功但無法執行（Permission denied）： # 加入執行權限（Linux/macOS） chmod +x hello # 如果出現「undefined reference」錯誤： # 可能需要連結數學函式庫 gcc -Wall hello.c -o hello -lm
編譯時常見錯誤：
  • 語法錯誤（Syntax Error）：檢查是否漏了分號、括號不匹配等
  • 未宣告的識別字：檢查變數或函數是否已宣告
  • 型態不符：檢查資料型態是否正確
  • 找不到標頭檔：檢查 #include 是否正確
  • 連結錯誤：檢查是否所有函數都有實作
除錯建議：
  • 仔細閱讀錯誤訊息，通常會指出問題所在的行號
  • 使用 -Wall -Wextra 編譯選項，查看所有警告
  • 一次修改一個問題，重新編譯確認
  • 善用 printf 輸出中間結果進行除錯

3. 第一個C程式

Hello World 程式

#include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }

程式詳細解析

// ========== 第一行：預處理指令 ========== #include <stdio.h> // #include：預處理器指令，告訴編譯器引入標頭檔 // <stdio.h>：Standard Input Output Header // 標準輸入輸出標頭檔 // 包含 printf、scanf 等函數的宣告 // 必須寫在程式開頭 // ========== 主函數 ========== int main() { // int：函數的返回型態為整數 // main：主函數，程式的進入點 // ()：參數列表（這裡沒有參數） // {：函數主體開始 // ========== 輸出敘述 ========== printf("Hello, World!\n"); // printf：輸出函數，定義在 stdio.h // "Hello, World!"：要輸出的字串 // \n：跳脫字元，代表換行 // ;：敘述結尾的分號（必須） // ========== 返回敘述 ========== return 0; // return：返回值給作業系統 // 0：表示程式正常結束 // 非零值：表示程式異常結束 // }：函數主體結束 }

程式執行流程

  1. 預處理階段：處理 #include 指令，將 stdio.h 的內容插入程式
  2. 編譯階段：將C程式碼轉換成組合語言，再轉換成機器碼
  3. 連結階段：將程式碼與所需的函式庫連結在一起
  4. 執行階段：
    • 作業系統載入程式到記憶體
    • 找到 main 函數並開始執行
    • 執行 printf 輸出文字到螢幕
    • 執行 return 0 返回給作業系統
    • 程式結束
重要觀念：
  • #include <stdio.h> 必須在程式開頭
  • main() 是程式的起點，每個C程式都需要一個 main 函數
  • 每個敘述結尾要加分號 ;
  • 大括號 { } 包含函數的主體，標記程式碼區塊
  • return 0 表示程式正常執行完畢（慣例）
  • C語言對大小寫敏感：Main 和 main 是不同的
  • 縮排不是強制的，但為了可讀性應該適當縮排

常見的跳脫字元

跳脫字元 意義 範例
\n 換行（New Line） printf("Hello\nWorld");
\t 水平定位（Tab） printf("Name:\tJohn");
\\ 反斜線 printf("C:\\Program Files");
\" 雙引號 printf("He said \"Hi\"");
\' 單引號 printf("It\'s OK");
\r 歸位（Carriage Return） printf("Loading...\r");
\b 退回一格（Backspace） printf("ABC\bD");
\0 空字元（Null） 字串結束標記

更多 Hello World 變化

#include <stdio.h> int main() { // 基本輸出 printf("Hello, World!\n"); // 多行輸出 printf("Hello,\n"); printf("World!\n"); // 一次輸出多行 printf("Line 1\nLine 2\nLine 3\n"); // 使用 Tab 對齊 printf("Name:\tJohn\n"); printf("Age:\t20\n"); // 輸出特殊字元 printf("Path: C:\\Program Files\\App\n"); printf("He said, \"Hello!\"\n"); // 裝飾性輸出 printf("***********************\n"); printf("* Hello, World! *\n"); printf("***********************\n"); return 0; }

為什麼需要資料型態？

資料型態告訴編譯器：

  • 需要多少記憶體空間
  • 如何解釋這段記憶體的內容
  • 可以對這個資料進行哪些操作

變數宣告與初始化

#include <stdio.h> int main() { // ========== 變數宣告 ========== int age; // 宣告整數變數 float height; // 宣告浮點數變數 char grade; // 宣告字元變數 // ========== 變數賦值 ========== age = 20; height = 175.5; grade = 'A'; // ========== 宣告時初始化（推薦做法）========== int score = 95; double pi = 3.14159; char initial = 'J'; // ========== 同時宣告多個變數 ========== int x, y, z; // 宣告但未初始化 int a = 1, b = 2, c = 3; // 宣告並初始化 // ========== 不同型態的變數 ========== unsigned int count = 100; long long bigNumber = 9223372036854775807; float temperature = 36.5f; // f 表示 float 字面值 double precise = 3.141592653589793; // ========== 輸出變數值 ========== printf("Age: %d\n", age); printf("Height: %.1f cm\n", height); printf("Grade: %c\n", grade); printf("Score: %d\n", score); printf("Pi: %.5f\n", pi); return 0; }

變數命名規則

必須遵守的規則：

  • 只能包含字母、數字、底線（_）
  • 不能以數字開頭
  • 不能使用C語言保留字
  • 區分大小寫（age 和 Age 是不同的變數）

建議的命名慣例：

  • 使用有意義的名稱：score 比 s 好
  • 變數名稱用小寫開頭：studentAge
  • 常數用全大寫：MAX_SIZE
  • 多個單字可用：駝峰式（studentName）或底線（student_name）
  • 避免使用單一字母（除非在迴圈中：i, j, k）
// ========== 有效的變數名稱 ========== int age; int studentAge; int student_age; int age2; int _private; int MAX_VALUE; // ========== 無效的變數名稱 ========== // int 2age; // 不能以數字開頭 // int student-age; // 不能包含 - // int int; // 不能使用保留字 // int my age; // 不能包含空格

常數定義

#include <stdio.h> // 方法一：使用 #define（預處理器宏） #define PI 3.14159 #define MAX_STUDENTS 100 int main() { // 方法二：使用 const 關鍵字 const int MAX_AGE = 120; const float GRAVITY = 9.8; // 使用常數 double circumference = 2 * PI * 10; printf("Pi = %.5f\n", PI); printf("Max students = %d\n", MAX_STUDENTS); printf("Circumference = %.2f\n", circumference); // MAX_AGE = 130; // 錯誤！常數不能修改 return 0; }
常見錯誤：
  • 使用未初始化的變數：會得到不確定的值（垃圾值）
  • 變數名稱拼錯：C語言區分大小寫
  • 型態不符：將浮點數賦值給整數會截斷小數部分
  • 超出範圍：數值超過該型態能表示的範圍
  • 重複宣告：在同一作用域內不能宣告同名變數

型態轉換

#include <stdio.h> int main() { // ========== 隱式轉換（自動轉換）========== int a = 10; double b = a; // int 自動轉為 double printf("b = %.1f\n", b); // 輸出 10.0 // ========== 顯式轉換（強制轉換）========== double x = 3.7; int y = (int)x; // 強制轉為 int，小數部分被截斷 printf("x = %.1f, y = %d\n", x, y); // x=3.7, y=3 // ========== 整數除法問題 ========== int m = 5, n = 2; printf("%d / %d = %d\n", m, n, m / n); // 2（整數除法） // 解決方案：將其中一個轉為浮點數 printf("%d / %d = %.2f\n", m, n, (double)m / n); // 2.50 return 0; }

5. 運算符

算術運算符

#include <stdio.h> int main() { int a = 10, b = 3; // 基本算術運算 printf("加法: %d + %d = %d\n", a, b, a + b); // 13 printf("減法: %d - %d = %d\n", a, b, a - b); // 7 printf("乘法: %d * %d = %d\n", a, b, a * b); // 30 printf("除法: %d / %d = %d\n", a, b, a / b); // 3（整數除法） printf("餘數: %d %% %d = %d\n", a, b, a % b); // 1 // 浮點數除法 double x = 10.0, y = 3.0; printf("\n浮點除法: %.2f / %.2f = %.2f\n", x, y, x / y); // 3.33 // 混合運算 int result = a + b * 2; // 先乘後加：10 + 6 = 16 printf("\n混合運算: %d + %d * 2 = %d\n", a, b, result); // 使用括號改變優先級 int result2 = (a + b) * 2; // 先加後乘：13 * 2 = 26 printf("使用括號: (%d + %d) * 2 = %d\n", a, b, result2); return 0; }

遞增與遞減運算符

#include <stdio.h> int main() { // ========== 後置遞增（x++）========== int x = 5; printf("x 初始值 = %d\n", x); // 5 printf("x++ 返回值 = %d\n", x++); // 5（先使用再遞增） printf("x 遞增後 = %d\n\n", x); // 6 // ========== 前置遞增（++x）========== int y = 5; printf("y 初始值 = %d\n", y); // 5 printf("++y 返回值 = %d\n", ++y); // 6（先遞增再使用） printf("y 遞增後 = %d\n\n", y); // 6 // ========== 後置遞減（x--）========== int m = 10; printf("m-- = %d, m = %d\n", m--, m); // 10, 9 // ========== 前置遞減（--x）========== int n = 10; printf("--n = %d, n = %d\n", --n, n); // 9, 9 // ========== 實際應用範例 ========== int count = 0; count++; // count = 1 count++; // count = 2 count++; // count = 3 printf("\n計數器最終值: %d\n", count); return 0; }

6. 輸入輸出

printf 格式化輸出

#include <stdio.h> int main() { int age = 20; float height = 175.5; double pi = 3.14159265359; char grade = 'A'; char name[] = "John"; // ========== 基本格式符 ========== printf("整數: %d\n", age); printf("浮點數: %f\n", height); printf("字元: %c\n", grade); printf("字串: %s\n\n", name); // ========== 控制小數位數 ========== printf("%.0f\n", height); // 176（四捨五入） printf("%.1f\n", height); // 175.5 printf("%.2f\n", height); // 175.50 printf("%.5f\n\n", pi); // 3.14159 // ========== 控制輸出寬度 ========== printf("|%5d|\n", age); // | 20|（右對齊，寬度5） printf("|%-5d|\n", age); // |20 |（左對齊，寬度5） printf("|%05d|\n\n", age); // |00020|（補零） // ========== 組合使用 ========== printf("|%8.2f|\n", height); // | 175.50|（寬度8，2位小數） printf("|%-8.2f|\n\n", height); // |175.50 |（左對齊） // ========== 多個變數輸出 ========== printf("姓名: %s, 年齡: %d, 等級: %c\n", name, age, grade); // ========== 不同進制輸出 ========== int num = 255; printf("十進制: %d\n", num); // 255 printf("八進制: %o\n", num); // 377 printf("十六進制: %x\n", num); // ff printf("十六進制: %X\n", num); // FF return 0; }

格式符 說明 範例
%d, %i 有符號十進制整數 printf("%d", 123);
%u 無符號十進制整數 printf("%u", 123);
%f 浮點數（預設6位小數） printf("%f", 3.14);
%e, %E 科學記號 printf("%e", 1000.0);
%c 字元 printf("%c", 'A');
%s 字串 printf("%s", "Hello");

%p 指標位址 printf("%p", &var);

scanf 格式化輸入

#include <stdio.h> int main() { int age; float height; char grade; char name[50]; // ========== 讀取整數 ========== printf("請輸入年齡: "); scanf("%d", &age); // ========== 讀取浮點數 ========== printf("請輸入身高: "); scanf("%f", &height); // ========== 讀取字元 ========== printf("請輸入等級: "); scanf(" %c", &grade); // 注意 %c 前有空格 // ========== 讀取字串 ========== printf("請輸入姓名: "); scanf("%s", name); // 注意：name 前不加 & // ========== 一次讀取多個值 ========== int day, month, year; printf("請輸入日期 (日 月 年): "); scanf("%d %d %d", &day, &month, &year); // ========== 輸出結果 ========== printf("\n===== 輸入資料 =====\n"); printf("姓名: %s\n", name); printf("年齡: %d 歲\n", age); printf("身高: %.1f cm\n", height); printf("等級: %c\n", grade); printf("日期: %d/%d/%d\n", day, month, year); return 0; }
scanf 注意事項：
  • 必須加 & 符號：除了字串陣列外，變數前要加 &（位址運算符）
  • 格式符號要對應：整數用 %d，浮點數用 %f 或 %lf
  • 字元輸入問題：%c 會讀取包括換行符在內的所有字元，建議前面加空格 " %c"
  • 字串不會讀取空格：scanf("%s") 遇到空格就停止
  • 輸入緩衝區問題：多次輸入時可能需要清空緩衝區
常見錯誤：
  • 忘記加 & 符號：scanf("%d", age); ❌ 應為 scanf("%d", &age); ✓
  • 格式不符：輸入 "abc" 給整數變數會導致錯誤
  • 輸入超過陣列大小：字串會導致緩衝區溢位

getchar() 和 putchar()

#include <stdio.h> int main() { char ch; printf("請輸入一個字元: "); ch = getchar(); // 讀取單一字元 printf("你輸入的是: "); putchar(ch); // 輸出單一字元 printf("\n"); return 0; }

編碼風格建議：
  • 縮排：使用4個空格或1個Tab（統一使用）
  • 命名：使用有意義的變數名稱
  • 空格：運算符兩側加空格
  • 括號：大括號獨立一行或與語句同行（保持一致）
  • 註解：適當添加註解說明複雜邏輯
  • 空行：邏輯區塊之間加空行分隔
  • 行長度：每行不超過80個字元

8. 常見錯誤與除錯

編譯時錯誤

// 錯誤 1：忘記分號 int x = 10 // ❌ 缺少分號 int x = 10; // ✓ 正確 // 錯誤 2：括號不匹配 if (x > 5 { // ❌ 缺少右括號 if (x > 5) { // ✓ 正確 // 錯誤 3：未宣告變數 y = 10; // ❌ y 未宣告 int y = 10; // ✓ 正確 // 錯誤 4：型態不符 int num = "123"; // ❌ 字串不能直接賦值給int int num = 123; // ✓ 正確

執行時錯誤

// 錯誤 1：除以零 int a = 10, b = 0; int c = a / b; // ❌ 除數為零 // 正確做法：檢查除數 if (b != 0) { int c = a / b; } else { printf("錯誤：除數不能為零！\n"); } // 錯誤 2：陣列越界 int arr[5]; arr[10] = 100; // ❌ 陣列索引超出範圍 // 正確做法：確保索引在範圍內 if (10 < 5) { arr[10] = 100; } // 錯誤 3：使用未初始化的變數 int x; printf("%d", x); // ❌ x 未初始化，會得到垃圾值 // 正確做法：先初始化 int x = 0; printf("%d", x); // ✓ 正確

邏輯錯誤

// 錯誤 1：使用 = 而非 == if (x = 5) { // ❌ 這是賦值，不是比較！ // ... } if (x == 5) { // ✓ 正確的比較 // ... } // 錯誤 2：整數除法 int a = 5, b = 2; double result = a / b; // ❌ 結果是 2.0，不是 2.5 // 正確做法：先轉換為浮點數 double result = (double)a / b; // ✓ 結果是 2.5 // 錯誤 3：分號錯誤位置 if (x > 5); // ❌ 這裡的分號會讓 if 無效 { printf("x 大於 5"); // 這行永遠會執行 } if (x > 5) { // ✓ 正確 printf("x 大於 5"); }
除錯技巧：
  • 仔細閱讀錯誤訊息：編譯器會告訴你問題在哪一行
  • 使用 printf 除錯：在關鍵位置輸出變數值
  • 分段測試：將程式分成小部分逐一測試
  • 使用除錯器：GDB 等工具可以逐步執行程式
  • 檢查邊界條件：測試極端值（0、負數、最大值）
  • 保持程式簡潔：複雜的程式碼容易出錯

9. 完整範例程式

範例一：簡易計算機

#include <stdio.h> int main() { double num1, num2, result; char operator; printf("===== 簡易計算機 =====\n"); // 輸入第一個數字 printf("請輸入第一個數字: "); scanf("%lf", &num1); // 輸入運算符號 printf("請輸入運算符號 (+, -, *, /): "); scanf(" %c", &operator); // 輸入第二個數字 printf("請輸入第二個數字: "); scanf("%lf", &num2); // 計算結果 if (operator == '+') { result = num1 + num2; } else if (operator == '-') { result = num1 - num2; } else if (operator == '*') { result = num1 * num2; } else if (operator == '/') { if (num2 != 0) { result = num1 / num2; } else { printf("錯誤：除數不能為零！\n"); return 1; } } else { printf("錯誤：無效的運算符號！\n"); return 1; } // 輸出結果 printf("\n結果: %.2f %c %.2f = %.2f\n", num1, operator, num2, result); return 0; }

範例二：溫度轉換

#include <stdio.h> int main() { double celsius, fahrenheit; int choice; printf("===== 溫度轉換程式 =====\n"); printf("1. 攝氏轉華氏\n"); printf("2. 華氏轉攝氏\n"); printf("請選擇 (1 或 2): "); scanf("%d", &choice); if (choice == 1) { printf("請輸入攝氏溫度: "); scanf("%lf", &celsius); fahrenheit = (celsius * 9.0 / 5.0) + 32.0; printf("%.2f°C = %.2f°F\n", celsius, fahrenheit); } else if (choice == 2) { printf("請輸入華氏溫度: "); scanf("%lf", &fahrenheit); celsius = (fahrenheit - 32.0) * 5.0 / 9.0; printf("%.2f°F = %.2f°C\n", fahrenheit, celsius); } else { printf("無效的選擇！\n"); return 1; } return 0; }

範例三：BMI 計算器

#include <stdio.h> int main() { double height_cm, weight_kg, height_m, bmi; printf("===== BMI 計算器 =====\n"); // 輸入身高（公分） printf("請輸入身高（公分）: "); scanf("%lf", &height_cm); // 輸入體重（公斤） printf("請輸入體重（公斤）: "); scanf("%lf", &weight_kg); // 將身高轉換為公尺 height_m = height_cm / 100.0; // 計算 BMI = 體重(kg) / 身高(m)² bmi = weight_kg / (height_m * height_m); // 輸出結果 printf("\n===== 計算結果 =====\n"); printf("身高: %.1f 公分 (%.2f 公尺)\n", height_cm, height_m); printf("體重: %.1f 公斤\n", weight_kg); printf("BMI: %.2f\n", bmi); return 0; }

本週練習題

基礎練習（必做）

  1. 個人資訊輸出
    撰寫程式，宣告並初始化你的姓名、年齡、身高、體重等變數，然後以格式化方式輸出。
  2. 圓面積計算
    讀取圓的半徑，計算並輸出圓的面積和周長。（π = 3.14159）
    公式：面積 = πr²，周長 = 2πr
  3. 平均分數計算
    讀取三科成績，計算並輸出平均分數。

進階練習

  1. BMI 計算器
    讀取身高（公分）和體重（公斤），計算 BMI 值。
    公式：BMI = 體重(kg) / (身高(m))²
  2. 時間換算
    讀取總秒數，轉換成「小時:分鐘:秒數」格式輸出。
    例如：3665 秒 → 1 小時 1 分鐘 5 秒
  3. 複利計算
    讀取本金、年利率、存款年數，計算本利和。
    公式：本利和 = 本金 × (1 + 年利率)^年數
    提示：可以使用 pow() 函數，需要 #include <math.h>

挑戰練習

  1. 單位轉換程式
    製作一個選單式程式，可以進行以下轉換：
    • 公里 ↔ 英里（1 英里 = 1.60934 公里）
    • 公斤 ↔ 磅（1 磅 = 0.453592 公斤）
    • 攝氏 ↔ 華氏
  2. 購物結帳系統
    讀取商品單價和數量，計算小計。
    然後讀取折扣率（如 0.9 表示九折），計算折扣後金額。
    最後讀取顧客付款金額，計算找零。
