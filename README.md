# C語言程式設計課程

完整的16週C語言學習教材，從基礎到進階的系統化課程。

## 課程簡介

本課程提供完整的C語言學習路徑，涵蓋從基礎語法到進階應用的所有核心主題。適合程式設計初學者、資訊科系學生，以及需要紮實掌握C語言的學習者。

## 如何使用

1. 在瀏覽器中開啟 `index.html`
2. 從 Week 1 開始按順序學習
3. 完成每週的練習題
4. 實際撰寫並執行程式碼

## 課程結構

### 第一階段：基礎入門 (Week 1-4)
- Week 1: 開發環境與基礎語法
- Week 2: 流程控制
- Week 3: 陣列與字串
- Week 4: 函數設計

### 第二階段：核心進階 (Week 5-8)
- Week 5: 指標基礎
- Week 6: 動態記憶體管理
- Week 7: 結構體與自定義型態
- Week 8: 檔案輸入輸出

### 第三階段：高階特性 (Week 9-12)
- Week 9: 預處理器與巨集
- Week 10: 進階指標應用
- Week 11: 進階字串處理
- Week 12: 資料結構實作

### 第四階段：實踐應用 (Week 13-16)
- Week 13: 演算法設計與分析
- Week 14: 實用工具開發
- Week 15: 綜合專案開發 (一)
- Week 16: 綜合專案開發 (二)

## 學習建議

- **每週投入時間**: 7-10 小時
- **學習方式**: 理論學習 + 程式實作
- **重點**: 務必實際撰寫程式碼，不要只看不練
- **進度**: 可依個人情況調整，但建議按順序學習

## 環境需求

### Windows
- Visual Studio Community / Code::Blocks / MinGW + VS Code

### Linux
```bash
sudo apt install build-essential
```

### macOS
```bash
xcode-select --install
```

## 編譯執行

```bash
# 編譯
gcc program.c -o program

# 執行
./program        # Linux / macOS
program.exe      # Windows
```

## 學習成果

完成本課程後，你將能夠：
- 紮實掌握C語言核心語法
- 深入理解指標與記憶體管理
- 實作常見資料結構與演算法
- 開發實用的系統工具程式
- 完成完整的專案開發
